# 10. Авторське право та ліцензування

- Здобувач несе відповідальність за правомірність використання згенерованих матеріалів. Пам'ятайте про ризик **"успадкування ліцензії"**. Модель, навчена на коді з ліцензією GPL, може його відтворити. Ваша відповідальність — перевіряти походження згенерованих фрагментів.
- Для коду: вкажіть ліцензію (MIT/Apache‑2.0 тощо), джерела бібліотек; у коментарях позначайте місця, де допомагав ШІ (`// Generated with LLM, substantially modified`).
- Для медіа: перевіряйте умови використання/атрибуції; у звіті — джерело/модель/дата генерації.
