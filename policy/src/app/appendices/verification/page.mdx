# Додаток В. Чек-лист верифікації згенерованого коду

- **Коректність:** код компілюється/інтерпретується без помилок?
- **Надійність:** проходить усі наявні тести? Покриває edge-cases?
- **Безпека:** не містить очевидних вразливостей (SQL-ін'єкції, XSS)? Пройдено перевірку статичним аналізатором (SAST)?
- **Актуальність:** не використовує застарілі (deprecated) або небезпечні бібліотеки?
- **Продуктивність:** чи є прийнятною для поставленого завдання?
- **Обробка помилок:** чи коректно обробляються виняткові ситуації?

**Security-чек-лист:**

- [ ]  SAST пройдений;
- [ ]  secret-scan (жодних ключів/паролів у репозиторії);
- [ ]  залежності без критичних CVE;
- [ ]  SBOM (за можливості).
