# 3. Теоретичне підґрунтя

- **Педагогічна доцільність:** ШІ як **scaffolding tool** (інтелектуальна опора) для подолання розриву між поточним і потенційним рівнем оволодіння, що дозволяє виконувати завдання на вищому когнітивному рівні.
- **Таксономія Блума:** використання ШІ для завдань високого порядку (аналіз, створення, оцінка), а не для простого відтворення інформації.
- **Академічна доброчесність:** превенція недоброчесності через культуру прозорості, декларування та відтворюваності процесу роботи.
- **Компетентності майбутнього:** розвиток промпт-інжинірингу, критичної валідації результатів, етичної рефлексії та метапізнання (усвідомлення власного процесу навчання).
