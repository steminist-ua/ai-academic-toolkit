**Приклад 4. Бази даних (SQL)**

```markdown
ІНСТРУМЕНТИ: GPT-4o (ChatGPT, 2025-09-14), Copilot (DataGrip).

МЕТА: Підготовка стартового SQL-запиту для створення таблиці Orders із зовнішнім ключем на Customers.

ПРОМПТИ:
	1.	«Напиши SQL-DDL для створення таблиці Orders з атрибутами order_id, customer_id, status, total_amount.»
	2.	«Додай зовнішній ключ на таблицю Customers та значення за замовчуванням для статусу new.»

ВЛАСНИЙ ВНЕСОК: Код був адаптований під PostgreSQL (типи SERIAL, NUMERIC(10,2)), додано перевірку обмежень (CHECK status IN (‘new’,‘shipped’,‘cancelled’)), індексація за customer_id.

ВЕРИФІКАЦІЯ: Виконано тестові вставки/видалення, додано 5 запитів SELECT для edge-cases, проведено EXPLAIN ANALYZE для продуктивності.

РЕФЛЕКСІЯ: ШІ згенерував некоректний синтаксис для дефолтного значення (DEFAULT = 'new'), виправлено вручну. Також довелося додати індекси для швидкості JOIN-ів.

```
